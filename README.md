# Discuz! Q Framework

基于 Laravel Illuminate 组件的论坛系统框架。

## 安装

```bash
composer require mshk/core
```

## 系统要求

- PHP >= 8.2
- ext-json
- ext-mbstring
- ext-openssl

## 主要组件

- 用户认证与授权
- 帖子与评论系统
- 支付系统 (微信支付)
- 文件存储 (本地/COS)
- 缓存系统 (文件/Redis)
- 队列系统 (Redis)
- 短信服务
- 微信集成
